## Technology

- Python3: Flask and Bottle framework
- NGINX: CLIENT-SIDE CERTIFICATE AUTHENTICATION
- IAM: Keycloak. Includes keycloak.js
- MQTT: Eclipse Mosquitto
- Docker: docker and docker-compose plugin
